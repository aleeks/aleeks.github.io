monfrigo.fr

Projet universitaire : L'entreprise « Mon frigo facile » (une entreprise factice définie pour ce projet dont Nicolas Côté est le patron) lutte contre le gaspillage alimentaire dans le monde, c'est une entreprise française en pleine expansion.
L'entreprise souhaite faire prendre conscience que l'on peut éviter le gaspillage alimentaire. Elle désire donc se doter d'un site internet permettant à un utilisateur de gérer un réfrigérateur virtuel (réfrigérateur qui n'est autre qu'une métaphore pour ranger tous les aliments d'un utilisateur). L'entreprise souhaite également proposer des conseils pour éviter le gaspillage alimentaire.

  • Date : Mars - Juin 2014
  • Technologies : CakePHP, JQuery, Bootstrap

Maquettes Terre d'Aventure
